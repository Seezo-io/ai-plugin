# Seezo AI Plugin

Use the Seezo skills in this repository for Seezo-assisted security work:

- `plugins/seezo/skills/start-assessment/SKILL.md`
- `plugins/seezo/skills/validate-implementation/SKILL.md`

Keep outputs evidence-led. Separate confirmed facts, assumptions, risks, recommendations, and open questions. Use the Seezo MCP only when it is configured, authenticated, and the available tool descriptions match the task.

Never expose secrets, customer data, private architecture details, internal Seezo prompts, proprietary evaluator logic, or unpublished scoring rules. If MCP is unavailable or unauthenticated, continue without MCP and mark MCP evidence as unavailable.
